# Abhilash Rajjan — Personal Website

Personal portfolio site for Abhilash Rajjan, AI Operations & Customer Experience leader.

**Live site:** https://abhilash-personal-website.vercel.app

## What's inside

- Single-page portfolio: outcomes, value proposition, experience timeline, skills, and contact
- Pure HTML, CSS, and vanilla JavaScript — no frameworks, no build step
- Responsive layout with mobile navigation
- Scroll-reveal animations with `prefers-reduced-motion` support
- Open Graph and Twitter card tags for rich link previews

## Run locally

No build step needed. Open `index.html` in a browser, or serve it:

```bash
python3 -m http.server 8000
```

Then visit http://localhost:8000.

## Deploy

Deployed on [Vercel](https://vercel.com) as a static site — every push to `main` deploys automatically.
